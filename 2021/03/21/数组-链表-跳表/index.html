70.爬楼梯

问题：

思路：动态规划

划分成子问题。

假设走 n - 1步需要f(n-1)

那么走f(n)就需要f(n-1)+f(n-2)

即递推公式：

f ( n ) = f ( n - 1 ) + f ( n - 2 )

其中初值 f ( 1 ) = 1, f ( 2) = 2.
